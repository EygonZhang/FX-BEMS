User Guide:

1. Newly create a working directory (e.g., the file path of ./XFoil input);

2. When using, ensure that the xfoil.exe file is copied into this working directory;

3. Should you revise the input values in Commands.txt, always remember to delete/remove the "output_myfoil.csv" first;  

4. Users can modify the target parameters in the Commands.txt file, which include: (1) airfoil data, (2) panel number (i.e. 1D mesh density), (3) Re, (4) ğ‘crit value, (5) number of iterations, and (6) dynamic twist angle. Detailed usage instructions are seen in the "Commands with full comments" for operation reference.
